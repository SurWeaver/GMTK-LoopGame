Меню -> Заходим на уровень
	Список уровней
	Изначально открыт один, все открываются последовательно после первого завершения

	Каждый уровень представляет собой сцену с несколькими рядами, откуда вылезают враги

	У уровня есть набор волн. "Администратор" уровня спавнит сразу всю волну на арену.
	Игрок должен зачистить всю волну. По поражению всех мишеней спавнится новая волна.
	Когда волн не осталось, уровень завершается.

	У каждой мишени есть своё количество "здоровья", т.е. мишень может быть поражена не с первого выстрела.



	Под каждым уровнем располагаются иконки с патронами (чёрный силуэт) и краткое описание для открытия
	На каждом уровне есть около 3 разблокируемых типов патронов
	Каждый разблокируется разными целями
		Примеры:
		- Количество очков
		- Время прохождения
		- Поражение скрытой цели
		- Выстрел в уязвимое место конкретной цели


Стреляем по мишеням
	За попадание по каждой мишени получаем количество очков
	За попадание в уязвимое место получаем увел. кол-во очков


Инфо по мишеням

	Дробь (увеличенная область попадания и урон)
	Снайпер (увел. множитель урона по уязвимым зонам)
	Дальнозоркость/Разбег/Разгон (Чем дальше враг, тем больше урон)
